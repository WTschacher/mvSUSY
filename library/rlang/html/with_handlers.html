with_handlers {rlang}R Documentation

Establish handlers on the stack

Description

As of rlang 1.0.0, with_handlers() is deprecated. Use the base functions or the experimental try_fetch() function instead.

Usage

calling(handler)

exiting(handler)

Arguments
