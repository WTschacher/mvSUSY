Description

Transformation for times (class hms)

Usage

hms_trans()

Examples

if (require("hms")) {
  hms <- round(runif(10) * 86400)
  t <- hms_trans()
  t$transform(hms)
  t$inverse(t$transform(hms))
  t$breaks(hms)
}
