mvsusy print method

Description

Prints information about an mvsusy object.

Usage

  ## S3 method for class 'mvsusy'
print(x, ...)

Arguments

x

A susy object.

...

Extra arguments passed to print.data.frame method.

Value

Returns x invisibly. Display output to console as a side effect.

mvsusy
