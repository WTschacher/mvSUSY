link_path {fs}R Documentation

Read the value of a symbolic link

Description

Read the value of a symbolic link

Usage

link_path(path)

Arguments

path

A character vector of one or more paths.

Value

A tidy path to the object the link points to.

Examples


file_create("foo")
link_create(path_abs("foo"), "bar")
link_path("bar")

# Cleanup
file_delete(c("foo", "bar"))
